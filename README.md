# 🏋️‍♂️ Gym Performance & Member Insights Dashboard

## 📊 Project Overview
This Power BI dashboard provides an end-to-end analytics solution for a fitness center. It transforms raw operational data into actionable insights regarding member attendance, revenue trends, membership types, and peak gym hours. 

The goal of this project is to help gym management optimize staffing, target marketing campaigns, and improve member retention.

## 💡 Key Insights & Features
* **Member Retention:** Tracked active vs. churned members to identify at-risk accounts.
* **Revenue Operations:** Analyzed monthly recurring revenue (MRR) broken down by membership tiers (e.g., VIP, Standard, Basic).
* **Attendance Tracking:** Visualized peak workout times and days to optimize staff scheduling and class availability.
* **Demographic Breakdown:** Created member personas based on age, gender, and frequency of visits.

---

## 🛠️ Tech Stack & Skills Demonstrated
* **Tool:** Power BI Desktop
* **Data Modeling:** Established a Star Schema utilizing Fact and Dimension tables.
* **DAX (Data Analysis Expressions):** Wrote advanced measures for rolling averages, Year-over-Year (YoY) growth, and active member counting.
* **Data Transformation:** Cleansed and shaped raw data using Power Query (M language), handling missing values and data type corrections.
